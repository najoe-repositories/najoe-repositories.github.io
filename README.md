# najoe-repositories.github.io

Cyberpunk single-page portfolio for **Jo (Najoe)** — SRE, DevOps, security research, fullstack, and WordPress. Jekyll on GitHub Pages.

**Live:** https://najoe-repositories.github.io

## Sections

| Anchor | Content |
|--------|---------|
| `#home` | Hero + portrait |
| `#about` | Profile & skills |
| `#portfolio` | Selected work (Najoe Solutions) |
| `#contact` | Hire / project form |

## Edit content

- `_data/profile.yml` — bio, skills, services, links
- `_data/portfolio.yml` — portfolio cards
- `_data/menus.yml` — nav / footer
- `images/jo-portrait.png` — hero portrait

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Open http://127.0.0.1:4000
